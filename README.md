# GPT2 Poetic Chatbot

## Introduction
This project implements a poetic chatbot using Hugging Face Transformers and DialoGPT.

## Features
- Generate poetic responses
- Interactive chatbot
- Uses pre-trained language model
- Real-time text generation

## Technologies Used
- Python
- Hugging Face Transformers
- DialoGPT
- Google Colab
